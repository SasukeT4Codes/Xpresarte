Sobre Xpresarte

Xpresarte es un prototipo para crear stickers personalizados a partir de un modelo modular. Este prototipo permite combinar capas, recolorear elementos y exportar imágenes y una hoja A4 para impresión.

  • Formato de trabajo: PNG modulados y recoloreables.
  • Exportes: PNG 512×512 para stickers y PDF A4 para impresión.
  • Tecnología: Django + Pillow (backend) y JavaScript (preview cliente).
